Файлы:

* Program.cs - Главное Меню. Основной файл. **Для его работоспособности, в той же директории необходимо наличие 4-х файлов с задачами:**

    * Zadacha1.cs - Код решения Задачи 1: на вход принимаются два целых числа и выдаётся большее и меньшее значения среди них.

    * Zadacha2.cs - Код решения Задачи 2: на вход принимаются три целых числа и выдаётся большее значение среди них.

    * Zadacha3.cs - Код решения Задачи 3: на вход принимается целое число и выдаётся, является ли оно чётным.

    * Zadacha4.cs - Код решения Задачи 4: на вход принимается целое число (N), а выдаются все чётные числа от 1 до (N) включительно.

**Код из файлов Задач - самостоятельный, предусмотрена работа и без Главного Меню.**
